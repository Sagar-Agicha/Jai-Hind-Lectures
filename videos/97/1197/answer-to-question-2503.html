Course creation can be done by Moodle site administrator/ Course Creator/ Manager.
Users assigned with 'Teacher' role in Moodle can only edit the courses assigned to them; and cannot create new courses.
They need to contact Site admin for this.

If you are 'Site administrator' in the Moodle platform you are using, then courses can be created as follows:
- Site administration -> Courses -> Manage courses and categories -> 'Create new course'
OR
- Site home -> 'Add a new course'

OR 
If the course is already created, then you should be enrolled as a 'Teacher' to that course.
Then the course will be displayed in your dashboard.
Again, if you have admin capabilities then, you can enrol yourself as a 'Teacher' to that course OR request the site admin to do so.

A few things you can try are:
