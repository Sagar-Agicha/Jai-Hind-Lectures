Answers:

When you say 'unable to do student login', what does the error message say?

Also, if it is forgotten username or password error for student login, then you can reset the student login credentials through your admin a/c:
- Login as admin
- Go to Site administration -> Users -> Browse list of users
- Here, you will find the student name in the list, click on the 'Edit' icon for that student
- You will find that particular student details like username and also, you can reset the password 
- Try logging in as that student using new credentials

Let me know if you have any further queries or issues.

Also, if you wish to get a student view, there is an option available:
- Login in as a teacher or admin user
- Click on your name (right hand side top corner, where you get an option to logout)
- You will find an option to 'Switch role to...', click on it
- Here, you will find a list of roles, click on 'Student'
- But, it does not give you an actual student view, some of the admin capabilities still persist

But, you can give it a try to take a look at this feature.
24-04-20, 12:02 p.m. kashmira.n

One more option available to admins is masquerading, i.e, 'Log in as' someone else

- Login as admin
- Open up Site administration -> Users -> Browse list of users
- Here, you get users listed out, click on the name of the student you wish to login as
- You will get to view his profile
- Here, under 'Administration' section, there is an option available called 'Log in as', click on it
- Click on 'Continue'
- You will be logged in as the student, and can get a view of Moodle setup from that particular students perspective, i.e, courses he's been enrolled into etc.
- Please do not change anything here, as your changes will get reflected in that users account 
- You can use this feature just to view your system from students point
