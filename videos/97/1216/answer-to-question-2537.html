Answers:

If you open up Question bank -> Import section, you will find a number of options available for importing quiz questions:
- Aiken format
- Blackboard
- Embedded answers (Cloze)
- Examview
- Gift format
- Missing word format
- Moodle XML format
- WebCT format

Check out Moodle docs related to this:
- https://docs.moodle.org/38/en/Import_questions

The simplest import option is the 'Aiken format' - 
- https://www.umass.edu/it/support/moodle/import-quiz-questions-moodle-using-aiken-format
