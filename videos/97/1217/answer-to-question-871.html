Answers:

there is Aiken format in Moodle. (Many formats are available but Aiken is easy)
Link for you reference: https://docs.moodle.org/36/en/Aiken_format
You can create questions, options in that format and import them directly.
Remember these questions will directly be imported in the question bank.
From the Question bank, you can use for any quiz. You can export these question bank category and reuse for another course or in next year.
