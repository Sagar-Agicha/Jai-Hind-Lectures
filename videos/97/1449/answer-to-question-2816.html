Please do the assignment specified in the tutorial.

Follow all the steps as specified in the tutorial. Make a note of the file names.

The teacher will add the feedback for each student's submission and save it in a different file. For example Assignment1-Q1-Feedback.odt. Create a zip file for all students submission.

Upload multiple feedback files in a zip will  enable teachers to send feedback files back to all students in one single action.
