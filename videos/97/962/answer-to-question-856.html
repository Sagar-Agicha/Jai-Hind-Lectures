Rubrics in Moodle
Can we customize our own Rubrics and automate its grading in Moodle?

Moodle-Learning-Management-System Overview-of-Moodle 00-01 min 0-10 sec 01-03-19, 12:15 p.m. jatinambasana

Answers:

Yes, you can.
When setting the activity, use Rubric or Marking guide as the grading option.
After set up, you are presented with a screen asking if a existing rubric be used or if you want a new one set up.  Select 'New', presuming you have not done this before.
You can now add criteria, marks etc.
You can access this later by using the Edit Setting option.
